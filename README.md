# Gaurav Portfolio (Modern Refresh)

A lightweight, trend-aware portfolio starter with:

- Glassmorphism-inspired UI and gradient accents
- Persistent dark/light mode toggle
- Filterable project cards
- Responsive layout and reduced-motion support
- Clean semantic HTML with no build step

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
